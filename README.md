# Cementarii

整合個人常用功能的《原神》資料庫＆工具平台

## 主要功能

- 角色資料查詢
- 武器資料查詢
- 聖遺物資料查詢
- 素材（僅角色和武器相關）資料查詢
- 實用工具
  - 樹脂計算器

*（部分角色、武器和聖遺物資料仍未匯入資料庫）*
